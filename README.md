## ‚ö° Core Modules

### 1. Command Center (Dashboard)
The central hub for your recovery journey.
*   **Hero Timer:** Real-time streak tracking (Days:Hours:Mins:Secs).
*   **Neural Ticker:** Live marquee displaying system status and affirmations.
*   **Telemetry Grid:** Visualizers for *Willpower*, *Focus*, and *Sync* levels.
*   **Rank System:** Auto-leveling based on streak duration (Initiate -> Ascendant).

### 2. Analytics Dashboard (NEW v2.2)
Full admin-style analytics suite for deep insights.
*   **Statistics Overview:** 4 key metric cards (Resets, Check-ins, Journals, Panic Uses).
*   **Streak Progression Chart:** 30-day line graph tracking daily streak values.
*   **Task Completion Chart:** Doughnut visualization of daily protocol adherence.
*   **Activity Timeline:** Real-time log of last 10 actions with timestamps.
*   **Insights Engine:** AI-style recommendations based on behavior patterns.
*   **Heatmap Calendar:** GitHub-style 20-week activity visualization with intensity levels.

### 3. Terminal CLI (Command Line Interface)
Interact with the system using power-user commands. Located in the bottom-right console.

| Command | Description |
| :--- | :--- |
| `help` | Display available commands. |
| `commit` | Execute daily check-in protocol. |
| `log <msg>` | Quick-save a journal entry to the Chronicles. |
| `panic` | Trigger Emergency Mode immediately. |
| `status` | Print current streak and rank statistics. |

### 4. Chronicles (Journaling System)
A persistent log to record your thoughts, triggers, and victories.
*   **Local Storage:** Data lives on your device. No cloud leaks.
*   **Quick Log:** Add entries via the dashboard or CLI.
*   **History:** Review past entries to identify patterns.

### 5. Emergency Protocol (Panic Button)
A "Break Glass" feature for high-urge moments.
*   **Visual Disengagement:** Full-screen overlay to block triggers.
*   **Breathing Guide:** 4-7-8 breathing animation to lower heart rate.
*   **Motivation:** Direct links to support communities and motivational content.

## üìä Analytics & Tracking

### What Gets Measured
*   **Streak History:** Daily snapshots for 140-day heatmap visualization
*   **Relapse Tracking:** Date and duration of each reset for pattern analysis
*   **Activity Log:** Timestamped events (check-ins, journals, panic uses, resets)
*   **Task Completion:** Per-task statistics across all daily protocols
*   **Best Streak:** Automatic tracking of your longest run

### Insights Engine
The system analyzes your data to provide:
*   Streak milestone celebrations (7+ days = top 10%)
*   Performance vs. personal average comparisons
*   Behavioral warnings (high panic button usage)
*   Consistency rewards (check-in milestones)

### Data Management
*   **Export:** Download complete backup as JSON with timestamp
*   **Import:** Restore from previous backup (with overwrite protection)
*   **Privacy:** All data stored locally in browser localStorage - zero cloud sync
